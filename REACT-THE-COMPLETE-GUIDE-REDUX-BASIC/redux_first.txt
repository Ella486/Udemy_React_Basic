<redux 처음>
node js 는 설치되어있다는 전제하에...
react가 아닌 일반 폴더일 때 

npm init -y
 ==> package.json 파일이 생김

npm install redux
 ==> package-lock.json 파일 생성되고 / node_modules 폴더 생성됨.

 demo.js 파일에 코드 작성 후 
 명령어 : node 파일이름.js 치면 작동됨.
 (ex. node redux-demo.js)

----------------------------------------------
git에서 다운로드 받고나서 파일을 실행할 떄
npm install
 ==> 끝나면 node 파일이름.js
